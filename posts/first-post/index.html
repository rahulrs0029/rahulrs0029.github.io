How to make a website like this?

Making a website like mine is very easy, thanks to Publish by John Sundell.

Here's how you can make a similar website.

Cloning the Publish repository from GitHub

Open Terminal (Search it in Spotlight or it is located in Applications > Utilities > Terminal.app)

Run these set of commands:

$ git clone https://github.com/JohnSundell/Publish.git
$ cd Publish
$ make

Next, run these another set of commands:

$ mkdir YourWebsiteName
$ cd YourWebsiteName
$ publish new

Finally, run open Package.swift. It will open Xcode. ...
